# Windows Subsystem for Linux
**List Installed Distros**
```
wsl -ls
```

**Install a Distro**
```
wsl --install ubuntu
```

**Set the Default Distro**
```
wsl --setdefault ubuntu
```

### Ubuntu
**Add a User to the Sudo Group**
```
bash
sudo usermod -aG sudo victor
```
