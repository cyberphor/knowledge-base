# Troubleshooting

## Network Issues
**How to View MTU Size**
```
netsh interface ipv4 show interfaces
```

**Misc Windows Commands**
```
ipconfig /all
ipconfig /release
ipconfig /renew
ipconfig /displaydns
ipconfig /flushdns
netstat
```

nslookup
